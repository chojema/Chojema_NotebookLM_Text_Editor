# 🎨 Chojema NotebookLM Text Editor

AI를 활용하여 이미지나 PDF의 텍스트를 지우고 내용을 수정하는 로컬 에디터입니다.
별도의 가입이나 API 키 없이 무료로 사용할 수 있습니다.

## 🚀 1분 만에 실행하기

### 윈도우(Windows) 사용자
1. 오른쪽 위 **Code** 버튼 -> **Download ZIP**을 눌러 다운로드합니다.
2. 압축을 풀고 폴더 안에 있는 **`run.bat`** 파일을 더블 클릭하세요.
   - 처음 한 번만 설치 과정(1~3분)이 진행되고, 두 번째부터는 바로 실행됩니다.

### 맥(Mac) 사용자
1. 터미널을 열고 다운로드 받은 폴더로 이동합니다.
2. 다음 명령어를 한 번만 입력하세요 (권한 설정):
   `chmod +x run.sh`
3. 실행합니다:
   `./run.sh`

## 🛠️ 필수 준비물
이 프로그램을 실행하려면 컴퓨터에 **Python(파이썬)**이 설치되어 있어야 합니다.
- [Python 공식 다운로드](https://www.python.org/downloads/)
- ⚠️ 설치 시 **"Add Python to PATH"** 체크박스를 반드시 체크해주세요!

## ✨ 주요 기능
- **PDF 지원:** Poppler 설치 없이 PDF 파일을 바로 열 수 있습니다.
- **AI 텍스트 제거:** PaddleOCR로 글자를 읽고, LaMa 모델로 배경을 복원합니다.
- **텍스트 추출:** 지워진 자리에 있던 텍스트를 자동으로 추출해줍니다.
