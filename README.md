# Chojema NotebookLM Text Editor

Google NotebookLM 등을 활용해 만든 발표용 슬라이드 PDF나 이미지의 텍스트를 지우고 내용을 수정하는 로컬 에디터입니다. 자신의 PC에 설치해 사용하는 것으로 별도의 가입이나 API 키 없이 무료로 사용할 수 있습니다.

## 설치 및 실행

### Windows
1. 오른쪽 위 **Code** 버튼 -> **Download ZIP**을 눌러 다운로드합니다.
2. 압축을 풀고 폴더 안에 있는 **`run.bat`** 파일을 더블 클릭하세요.
   - 처음 한 번만 설치 과정이 진행되면서 필요한 파일들이 다운로드됩니다. 이후로 바로 실행됩니다.
   - PC에 사용 중인 기본 웹 브라우저에 자동으로 열립니다. 자동으로 열리지 않으면 터미널에 표시된 주소를 사용하면 됩니다.

### macOS
1. Windows와 같은 방식으로 파일을 받아 압축을 풉니다.
1. 터미널을 열고 다운로드 받은 폴더로 이동합니다.
2. 다음 명령어를 입력합니다. 권한 설정을 위한 것으로 처음 한 번만 진행하면 됩니다.
   `chmod +x run.sh`
3. 실행합니다.
   `./run.sh`

## GPU 가속(선택 사항)
NVIDIA 그래픽 카드를 사용 중이라면, 속도 향상을 위해 GPU 가속을 켤 수 있습니다. CPU 모드보다 훨씬 빠릅니다.
1. 먼저 `run.bat`을 한 번 실행해서 기본 설치를 마칩니다.
2. 폴더 내의 **`install_gpu.bat`** 파일을 더블 클릭합니다.
   - 약 2.5GB 정도의 추가 파일을 다운로드합니다.
3. 설치가 끝나면 프로그램 사용은 `run.bat`으로 합니다. 자동으로 GPU를 감지하여 작동합니다.

## 필수 요구 사항
이 프로그램을 실행하려면 컴퓨터에 **Python**(파이썬)이 설치되어 있어야 합니다.
- [Python 공식 다운로드](https://www.python.org/downloads/)
- 설치할 때 **"Add Python to PATH"** 체크박스를 반드시 체크해주세요!

## 주요 기능
- **PDF 지원:** PDF 파일을 불러와 개별 페이지 단위로 작업하고 PNG 파일로 복사, 저장할 수 있습니다.
- **텍스트 추출:** 이미지에 있는 텍스트를 편집 가능한 상태로 추출합니다.
- **텍스트 제거 및 배경 복원:** 불필요한 텍스트나 대상물을 없애고 그 자리를 주변 배경과 어울리게 복원합니다.

## 주의 사항
- PowerPoint(파워포인트)나 기타 작업을 위한 애플리케이션을 열어 두고, 이 프로그램에서 수정 작업을 하면서 해당 이미지나 필요한 텍스트를 작업 중인 애플리케이션으로 복사해 사용하는 방식으로 진행하기를 추천합니다.
- 텍스트 추출이나 배경 복원 단추를 눌러도 단추 아래에 작업 중이라는 안내 메시지가 나오지 않고 제대로 작동하지 않는다면 잠깐 기다렸다 다시 시도해 보세요.
- 앱 설치 후 최초 실행 시에는 필요한 모델 다운로드나 메모리 탑재 등으로 시간이 좀 더 걸릴 수 있습니다.
- 작업 파일은 특정 서버로 전송되는 게 아니라 자신의 웹 브라우저 안에서 처리됩니다.
- 이 앱은 자동 업데이트되지 않으므로, 가끔 이 페이지를 방문하여 최신 버전이 있는지 확인하고 적용하기 바랍니다.
